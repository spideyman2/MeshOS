# MeshOS

MeshOS is an AI Operating System built for the Mesh API Hackathon.

## Features

- Modern AI dashboard
- Multi-agent architecture
- Planner
- Research
- Documents
- Report
- Prompt-based workflow
- Responsive UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- shadcn/ui
- Mesh API (Coming Day 2)
- Express.js (Coming Day 2)
